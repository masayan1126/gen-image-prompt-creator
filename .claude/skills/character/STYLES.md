# キャラクタースタイル定義

## 画風一覧

### 1. アニメ調
| 項目 | 設定 |
|------|-----|
| 特徴 | 日本アニメ風、目が大きめ |
| 等身 | 5-7等身 |
| 線 | はっきりした輪郭線 |
| 塗り | セルシェーディング、グラデーション |
| 適した用途 | イラスト、アイコン、ファンアート |

### 2. ゲーム調
| 項目 | 設定 |
|------|-----|
| 特徴 | ファンタジーRPG風、装備重視 |
| 等身 | 6-8等身 |
| 線 | 繊細な線、ディテール多め |
| 塗り | 厚塗り、質感表現 |
| 適した用途 | ゲームキャラ、ファンタジー系 |

### 3. ポップ調
| 項目 | 設定 |
|------|-----|
| 特徴 | カラフル、デフォルメ |
| 等身 | 2-4等身 |
| 線 | 太めの輪郭線 |
| 塗り | ビビッドカラー、フラット |
| 適した用途 | グッズ、ステッカー、SNS |

### 4. リアル調
| 項目 | 設定 |
|------|-----|
| 特徴 | 等身高め、詳細な描き込み |
| 等身 | 7-8等身 |
| 線 | 細い線、または線なし |
| 塗り | リアルな影、質感 |
| 適した用途 | ポートレート、コンセプトアート |

### 5. マスコット調
| 項目 | 設定 |
|------|-----|
| 特徴 | かわいい、シンプル、親しみやすい |
| 等身 | 2-3等身 |
| 線 | シンプルな輪郭 |
| 塗り | フラットカラー、単純 |
| 適した用途 | ロゴ、マスコット、ステッカー |

### 6. 手書き・ミニマル調
| 項目 | 設定 |
|------|-----|
| 特徴 | 手書き風ドゥードル、極限までシンプル、ステッカー風 |
| 等身 | 自由（シンプルな形状優先） |
| 線 | 黒インクの輪郭線、落書き風タッチ |
| 塗り | 純白のフラット塗り、影なし |
| 背景 | ソリッドなティール（青緑）単色 |
| 縁取り | 白い太めのステッカー風ボーダー |
| 適した用途 | ステッカー、アイコン、シンプルイラスト、グッズ |

### 7. スケッチ調

スケッチ調には4つのサブスタイルがあります。

#### 7-1. 鉛筆・デッサン風
| 項目 | 設定 |
|------|-----|
| 特徴 | クラシックなスケッチ、学術的・アーティスティックな印象 |
| 線 | グラファイト鉛筆の濃淡、2B〜6Bの太さ変化 |
| 塗り | ハッチング、クロスハッチングで陰影表現 |
| 色 | モノクロ基本、セピア調も可 |
| 質感 | 紙の質感、鉛筆のざらつき |
| 適した用途 | アート作品、観察スケッチ、教育素材、額装向け |

#### 7-2. 水彩スケッチ風
| 項目 | 設定 |
|------|-----|
| 特徴 | 柔らかく透明感のある表現、癒し系 |
| 線 | 薄い鉛筆の下書きライン、または線なし |
| 塗り | 水彩のにじみ、ウェットオンウェット、色の重なり |
| 色 | パステル調、透明感のある淡い色 |
| 質感 | 水彩紙のテクスチャ、絵の具の溜まり |
| 適した用途 | 絵葉書、ナチュラル系グッズ、癒し系アイコン、インテリア |

#### 7-3. ボールペン/万年筆風
| 項目 | 設定 |
|------|-----|
| 特徴 | 繊細な線画、クラシカルで知的な印象 |
| 線 | 一定の太さ、インクの濃淡、クロスハッチング |
| 塗り | 線のみで構成、塗りなし or 点描（スティップリング） |
| 色 | 黒、ブルーブラック、セピア |
| 質感 | 滑らかなペン運び、インクのにじみ |
| 適した用途 | 手帳イラスト、文房具グッズ、スタンプ、レターヘッド |

#### 7-4. クレヨン/色鉛筆風
| 項目 | 設定 |
|------|-----|
| 特徴 | 温かみのある手描き感、やわらかく親しみやすい印象 |
| 線 | ざらついた輪郭、不均一で味のある線 |
| 塗り | 重ね塗り、ストローク跡が残る、はみ出し許容 |
| 色 | 鮮やかだが柔らかい発色 |
| 質感 | 画用紙、クレヨンのワックス感、色鉛筆の木目 |
| 適した用途 | 子供向け、絵本風、温かみのあるグッズ、ポストカード |

## キャラクター要素

### 外見設定

#### 髪型・髪色
| カテゴリ | 例 |
|---------|-----|
| 長さ | ショート、ボブ、ミディアム、ロング、超ロング |
| スタイル | ストレート、ウェーブ、カール、ツインテール、ポニーテール |
| 色 | 黒、茶、金、銀、赤、青、紫、グラデーション |

#### 目
| カテゴリ | 例 |
|---------|-----|
| 形 | つり目、たれ目、丸目、切れ長 |
| 色 | 黒、茶、青、緑、金、赤、オッドアイ |
| 表現 | 輝く瞳、クール、優しい、鋭い |

#### 服装
| カテゴリ | 例 |
|---------|-----|
| カジュアル | Tシャツ、パーカー、ジーンズ |
| フォーマル | スーツ、ドレス、制服 |
| ファンタジー | 鎧、ローブ、魔法使いの衣装 |
| 職業系 | 白衣、エプロン、作業着 |

### 性格・雰囲気

| 性格 | 表現方法 |
|------|---------|
| 明るい | 笑顔、開いた姿勢、明るい色 |
| クール | 無表情、閉じた姿勢、寒色系 |
| 優しい | 柔らかい表情、暖色系 |
| 元気 | ダイナミックなポーズ、ビビッドカラー |
| ミステリアス | 影、目を隠す、ダークトーン |

### ポーズ

| 用途 | 推奨ポーズ |
|------|----------|
| アイコン | 正面、上半身、にっこり |
| 立ち絵 | 正面または斜め、全身、自然な立ち姿 |
| アクション | ダイナミックなポーズ、動きの表現 |
| 感情表現 | 喜怒哀楽に応じた体の動き |

### 表情

| 感情 | 表現 |
|------|-----|
| 喜び | 笑顔、目を細める、頬を赤らめる |
| 怒り | 眉をひそめる、歯を食いしばる |
| 悲しみ | 目を伏せる、涙、唇を噛む |
| 驚き | 目を見開く、口を開ける |
| 照れ | 視線をそらす、頬を赤らめる |
