# キャラクタースタイル定義

## 画風一覧

### 1. アニメ調
| 項目 | 設定 |
|------|-----|
| 特徴 | 日本アニメ風、目が大きめ |
| 等身 | 5-7等身 |
| 線 | はっきりした輪郭線 |
| 塗り | セルシェーディング、グラデーション |
| 適した用途 | イラスト、アイコン、ファンアート |

### 2. ゲーム調
| 項目 | 設定 |
|------|-----|
| 特徴 | ファンタジーRPG風、装備重視 |
| 等身 | 6-8等身 |
| 線 | 繊細な線、ディテール多め |
| 塗り | 厚塗り、質感表現 |
| 適した用途 | ゲームキャラ、ファンタジー系 |

### 3. ポップ調
| 項目 | 設定 |
|------|-----|
| 特徴 | カラフル、デフォルメ |
| 等身 | 2-4等身 |
| 線 | 太めの輪郭線 |
| 塗り | ビビッドカラー、フラット |
| 適した用途 | グッズ、ステッカー、SNS |

### 4. リアル調
| 項目 | 設定 |
|------|-----|
| 特徴 | 等身高め、詳細な描き込み |
| 等身 | 7-8等身 |
| 線 | 細い線、または線なし |
| 塗り | リアルな影、質感 |
| 適した用途 | ポートレート、コンセプトアート |

### 5. マスコット調
| 項目 | 設定 |
|------|-----|
| 特徴 | かわいい、シンプル、親しみやすい |
| 等身 | 2-3等身 |
| 線 | シンプルな輪郭 |
| 塗り | フラットカラー、単純 |
| 適した用途 | ロゴ、マスコット、ステッカー |

## キャラクター要素

### 外見設定

#### 髪型・髪色
| カテゴリ | 例 |
|---------|-----|
| 長さ | ショート、ボブ、ミディアム、ロング、超ロング |
| スタイル | ストレート、ウェーブ、カール、ツインテール、ポニーテール |
| 色 | 黒、茶、金、銀、赤、青、紫、グラデーション |

#### 目
| カテゴリ | 例 |
|---------|-----|
| 形 | つり目、たれ目、丸目、切れ長 |
| 色 | 黒、茶、青、緑、金、赤、オッドアイ |
| 表現 | 輝く瞳、クール、優しい、鋭い |

#### 服装
| カテゴリ | 例 |
|---------|-----|
| カジュアル | Tシャツ、パーカー、ジーンズ |
| フォーマル | スーツ、ドレス、制服 |
| ファンタジー | 鎧、ローブ、魔法使いの衣装 |
| 職業系 | 白衣、エプロン、作業着 |

### 性格・雰囲気

| 性格 | 表現方法 |
|------|---------|
| 明るい | 笑顔、開いた姿勢、明るい色 |
| クール | 無表情、閉じた姿勢、寒色系 |
| 優しい | 柔らかい表情、暖色系 |
| 元気 | ダイナミックなポーズ、ビビッドカラー |
| ミステリアス | 影、目を隠す、ダークトーン |

### ポーズ

| 用途 | 推奨ポーズ |
|------|----------|
| アイコン | 正面、上半身、にっこり |
| 立ち絵 | 正面または斜め、全身、自然な立ち姿 |
| アクション | ダイナミックなポーズ、動きの表現 |
| 感情表現 | 喜怒哀楽に応じた体の動き |

### 表情

| 感情 | 表現 |
|------|-----|
| 喜び | 笑顔、目を細める、頬を赤らめる |
| 怒り | 眉をひそめる、歯を食いしばる |
| 悲しみ | 目を伏せる、涙、唇を噛む |
| 驚き | 目を見開く、口を開ける |
| 照れ | 視線をそらす、頬を赤らめる |
