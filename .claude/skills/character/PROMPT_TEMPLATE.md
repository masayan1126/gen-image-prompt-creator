## 画風別テンプレート

### アニメ調

```
【画風】アニメ調
- 日本アニメ風のタッチ
- 目が大きめ、輝く瞳
- はっきりした輪郭線
- セルシェーディング
- 5-7等身

【キャラクター名】{name}

【外見設定】
- 性別・年齢: {gender_age}
- 髪: {hair}（アニメ的な表現、動きのある毛先）
- 目: {eyes}（大きく、ハイライト入り）
- 服装: {outfit}
- 体型: {body_type}

【性格・雰囲気】{personality}
【ポーズ】{pose}
【表情】{expression}
【背景】{background}

【補足指示】
- アニメのキャラクターデザイン風
- 鮮やかな色使い
{additional_notes}
```

```
{additional_notes}
```

```
- リアルな影と質感
- 詳細な描き込み

【キャラクター名】{name}

【外見設定】
- 性別・年齢: {gender_age}（リアルな年齢表現）
- 髪: {hair}（リアルな質感）
- 目: {eyes}（現実的な比率）
- 服装: {outfit}（布の質感、しわ）
- 体型: {body_type}（リアルな体型）

【性格・雰囲気】{personality}
【ポーズ】{pose}
【表情】{expression}（微細な表情）
【背景】{background}

【補足指示】
- コンセプトアート風の仕上がり
- 光と影のリアルな表現
{additional_notes}
```

### マスコット調

```
【画風】マスコット調
- かわいい、シンプル
- 2-3等身
- シンプルな輪郭
- フラットカラー
- 親しみやすいデザイン

【キャラクター名】{name}

【モチーフ】{motif}（動物、食べ物、物など）

【外見設定】
- ベースの形: {base_shape}（丸、楕円など）
- 色: {color}
- 特徴的な要素: {features}
- アクセント: {accent}

【性格・雰囲気】{personality}（一言で表現）
【ポーズ】{pose}（シンプルなポーズ）
【表情】{expression}（シンプルな表情）
【背景】{background}（シンプルまたは単色）

【補足指示】
- ロゴやマスコットとして使えるデザイン
- 単純化されているが個性的
{additional_notes}
```

### 手書き・ミニマル調

```
【画風】手書き・ミニマル調
- 手描き風ドゥードルスタイル
- 黒インクの輪郭線
- 純白のフラット塗りのみ
- 複雑なテクスチャや影は一切なし
- 物理的なステッカーのような太い白縁取り
- 極限までシンプルでクリーンなデザイン

【キャラクター/モチーフ】{subject}

【外見設定】
- 形状: {shape}（シンプルな形に簡略化）
- 特徴: {features}（最小限の要素で表現）
- 線のタッチ: 黒インク、落書き風
- 塗り: 純白のみ、グラデーションなし

【ポーズ】{pose}（シンプルなポーズ）
【表情】{expression}（シンプルな表情）

【レイアウト・構図】
- アスペクト比: 16:9
- 背景: ソリッドなティール（青緑）単色、グラデーションなし
- 主要オブジェクトの周りに太い白のステッカー風ボーダー
- テキスト、文字、副次的オブジェクトは配置しない

【補足指示】
- ステッカーとして切り抜けるようなデザイン
- 複雑な要素は排除し、本質的な形のみで表現
- プロフェッショナルかつクリエイティブな仕上がり
{additional_notes}
```

## バリエーション用テンプレート

### 表情差分

```
【ベースキャラクター】上記で作成したキャラクター

【表情バリエーション】
1. 笑顔（にっこり）
2. 怒り（眉をひそめる）
3. 悲しみ（涙目）
4. 驚き（目を見開く）
5. 照れ（頬を赤らめる）

【補足指示】
- 同一キャラクターの表情差分
- ポーズと服装は統一
- 表情以外の要素は変えない
```

### ポーズ差分

```
【ベースキャラクター】上記で作成したキャラクター

【ポーズバリエーション】
1. 正面立ち
2. 斜め立ち（3/4ビュー）
3. 座りポーズ
4. 走りポーズ
5. アクションポーズ

【補足指示】
- 同一キャラクターのポーズ差分
- 表情と服装は統一
- 体型とプロポーションを維持
```
