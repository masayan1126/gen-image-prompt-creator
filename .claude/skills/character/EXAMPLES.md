# サンプルプロンプト集

## 例1: アニメ調エンジニアキャラ

### プロンプト

```
【画風】アニメ調
- 日本アニメ風のタッチ
- 目が大きめ、輝く瞳
- はっきりした輪郭線
- セルシェーディング
- 6等身

【キャラクター名】テクノ

【外見設定】
- 性別・年齢: 男性、20代後半
- 髪: 黒髪ショート、少し寝癖がある、前髪が目にかかる
- 目: 青色、知的な印象、丸メガネをかけている
- 服装: 紺色のパーカー、白いTシャツ、ジーンズ
- 体型: 細身、普通体型

【性格・雰囲気】知的だが親しみやすい、少しオタク気質、コーヒー好き
【ポーズ】片手でノートPCを持ち、もう片手でコーヒーカップを持つ
【表情】自信のある笑顔、少し得意げ
【背景】シンプルな白背景、コードの断片が薄く浮かぶ

【アクセサリー・小物】
- 丸メガネ
- ワイヤレスイヤホン
- ステッカーだらけのノートPC
- コーヒーカップ

【補足指示】
- エンジニア・プログラマーの雰囲気
- 親しみやすいキャラクターデザイン
- プロフィールアイコンにも使えるデザイン
```

## 例2: ゲーム調ファンタジーキャラ

【表情】真剣な眼差し、少し微笑み

【アクセサリー・小物】
- 額に青い宝石のサークレット

- 指に魔力を込めたリング

【補足指示】
- ファンタジーゲームのキャラクター風

## 例3: ポップ調マスコットキャラ

### プロンプト

```
【画風】ポップ調
- カラフル、デフォルメ
- 2.5等身
- 太めの輪郭線
- ビビッドカラー、フラット塗り

【キャラクター名】コードくん

【外見設定】
- 性別・年齢: 不明（マスコット）、子供っぽい
- 髪: なし（ロボット風）
- 目: 大きな丸い目、LEDのように光る、青色
- 服装: なし（ロボット本体）
- 体型: 丸っこいロボット、手足は短い

【性格・雰囲気】元気いっぱい、好奇心旺盛、ちょっとドジ
【ポーズ】両手を上げてバンザイ、ジャンプ
【表情】にっこり笑顔、目がキラキラ
【背景】カラフルなグラデーション、星や光の粒子

【アクセサリー・小物】
- アンテナ（ハートマーク）
- 胸にコードのシンボル「</>」
- 足元にスニーカー風の装飾

【補足指示】
- テックブランドのマスコット風
- シンプルでグッズ化しやすいデザイン
- かわいくて親しみやすい
```

## 例5: マスコット調動物キャラ

### プロンプト

```
【画風】マスコット調
- かわいい、シンプル
- 2等身
- シンプルな輪郭
- フラットカラー

【キャラクター名】テクにゃん

【モチーフ】猫

【外見設定】
- ベースの形: 丸っこい猫、耳が三角
- 色: オレンジ（タビー柄）、白いお腹
- 特徴的な要素: 丸メガネ、猫耳が立っている
- アクセント: 尻尾の先が青く光っている

【性格・雰囲気】おっとり、でもプログラミングになると真剣
【ポーズ】座ってノートPCを見つめている
【表情】集中した顔、ちょっと舌が出ている
【背景】シンプルな白、キーボードのキーが周りに浮かんでいる

【アクセサリー・小物】
- 丸メガネ（ずれている）
- 小さなノートPC
- コーヒーカップ（猫の絵入り）

【補足指示】
- エンジニア向けSNSアカウントのマスコット
- シンプルでアイコンにしやすい
- 猫らしさとテック感のバランス
```

## 例6: アニメ調アイドルキャラ

### プロンプト

```
【画風】アニメ調
- アイドルアニメ風のタッチ
- キラキラした目、輝く瞳
- 鮮やかな色使い
- 6等身

【キャラクター名】星野ひかり

【外見設定】
- 性別・年齢: 女性、16歳
- 髪: ピンク色のツインテール、リボンで結んでいる、毛先がカール
- 目: 紫色、大きくて輝いている、星型のハイライト
- 服装: 白とピンクのアイドル衣装、フリルとリボン、ミニスカート
- 体型: 華奢、少女らしい体型

【性格・雰囲気】元気いっぱい、明るい、ちょっと天然
【ポーズ】ピースサイン、片足を上げてジャンプ
【表情】満面の笑顔、ウインク
【背景】ステージ風、スポットライト、キラキラのエフェクト

【アクセサリー・小物】
- 髪にピンクのリボン（2つ）
- 星形のイヤリング
- マイク
- ブレスレット

【補足指示】
- アイドルアニメ風のキャラクターデザイン
- キラキラエフェクト多め
- 明るくポップな雰囲気
```
