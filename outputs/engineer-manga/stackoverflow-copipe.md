# Stack Overflowからコピペ

> **重要**: このプロンプトを画像AIに入力する際は、キャラクターの元画像を一緒に添付してください。

## プロンプト

```
# 1. 役割と目標設定 (Role & Goal)
あなたは日本のプロ漫画家兼ストーリーボード演出家です。

以下のシナリオに基づき、4コマ構成のエンジニアあるある漫画を、1枚の高品質なカラー画像として描写してください。

【タイトル】Stack Overflowからコピペ（画像上部に大きく配置）

# 2. レイアウトとフォーマット (Layout & Format)
【形式・レイアウト】4コマ漫画、2x2グリッド配置
【読み順】左上→右上→左下→右下（Z字）
【枠線】黒い枠線と白い余白（solid black panel borders and white gutters）

# 3. キャラクター設定 (Character Bible)
添付画像のキャラクター（男の子とインコたち）を使用。
Use the attached character image as the reference for appearance, proportions, coloring, and emotional style.
Maintain strict character consistency across all panels.
外観・プロポーション・配色・表情のスタイルを全コマで厳密に一貫させてください。
男の子はデスクに向かうプログラマー、インコは肩や画面横でリアクション役として登場。

【背景設定】自宅のデスク、モニター、キーボード、暗めの部屋

# 4. シナリオ (Scenario)
## コマ1（起）
- 場面設定: 深夜の自室、モニターに赤いエラーメッセージ
- カメラ: ワイド（全体を見せる）
- 描写: 頭を抱えて困り果てた男の子、横でインコも心配そうな表情
- セリフ: 「なんでエラーなんだ...」
- 効果音: カタカタ...

## コマ2（承）
- 場面設定: 同じデスク、モニターにStack Overflowのページ
- カメラ: ミディアム（会話・反応）
- 描写: 画面を見て目が輝く男の子、インコも興味津々で画面を覗き込む
- セリフ: 「おっ、同じエラーの人がいる！」
- 効果音: ピコーン！

## コマ3（転）
- 場面設定: コードをコピペする瞬間
- カメラ: クローズアップ（表情強調）
- 描写: 無心でCtrl+C、Ctrl+Vする男の子、インコは首をかしげる
- セリフ: 「コピー...ペースト...」
- 効果音: カチャカチャ

## コマ4（結）
- 場面設定: 緑色の「Success」表示、でも困惑顔
- カメラ: 大アップ（インパクトの対比）
- 描写: 動いたけど理由がわからず複雑な表情の男の子、インコは「？」マーク
- セリフ: 「動いた...けどなんで動いたんだ...？」
- 効果音: ...

# 5. 演出とスタイル (Direction & Style)
【ジャンル・雰囲気】日常系ギャグ、ほのぼの、共感系
【画風】カラー、手書き風、シンプルな線画、パステル調
【線画】柔らかい線画、スケッチ調のタッチ
【演出効果】表情変化で感情を表現、コマ4の「動いたけど謎」感を強調

【補足指示】
- タイトル「Stack Overflowからコピペ」を画像上部に大きく配置
- 手書き風テイスト、スケッチ調のタッチ
- 柔らかい線で漫画らしい表現
- インコは各コマで男の子と一緒にリアクション

除外: 指が6本以上、不自然な体勢、コマごとの服装変化、顔が2つ
```

## 使用方法

1. 上記プロンプトをコピー
2. 画像生成AIにキャラクター画像を添付
3. プロンプトを入力して生成
4. 生成された画像を `outputs/engineer-manga/images/stackoverflow-copipe.png` に保存
