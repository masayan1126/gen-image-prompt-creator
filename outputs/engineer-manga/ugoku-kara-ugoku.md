# 動くから動くんです

> **重要**: このプロンプトを画像AIに入力する際は、キャラクターの元画像を必ず添付してください。

## プロンプト

```
# 1. 役割と目標設定 (Role & Goal)
あなたは日本のプロ漫画家兼ストーリーボード演出家です。

以下のシナリオに基づき、3コマ構成のエンジニアあるある漫画を、1枚の高品質なカラー画像として描写してください。

【タイトル】動くから動くんです（画像上部に大きく配置）

# 2. レイアウトとフォーマット (Layout & Format)
【形式・レイアウト】3コマ、縦配置、表情にフォーカス
【読み順】上から下へ
【枠線】黒い枠線と白い余白（solid black panel borders and white gutters）

# 3. キャラクター設定 (Character Bible) ※外観の一貫性が最重要
添付画像のキャラクター（男の子とインコたち）を使用。
Use the attached character image as the reference for appearance, proportions, coloring, and emotional style.
Maintain strict character consistency across all panels.
外観・プロポーション・配色・表情のスタイルを全コマで厳密に一貫させてください。

インコはリアクション役として登場し、男の子と一緒にシーンを盛り上げる。

【背景設定】シンプルな白背景、デスクとモニターのみ

# 4. シナリオ (Scenario)
## コマ1
- 場面設定: オフィスのデスク
- カメラ: ミディアム
- 描写: コードレビューのコメントを見て困惑した表情の男の子。肩にインコが1羽止まっている。
- セリフ: レビュアー「このコード、なぜ動くか説明してください」
- 効果音: なし

## コマ2
- 場面設定: 同じデスク
- カメラ: クローズアップ
- 描写: 顎に手を当てて考え込む男の子。インコも首を傾げて一緒に考えている様子。汗が一滴。
- セリフ: （...なぜだろう）
- 効果音: シーン...

## コマ3
- 場面設定: 同じデスク
- カメラ: ミディアム
- 描写: 開き直ったようなにっこり笑顔で両手を広げる男の子。インコも翼を広げてドヤ顔。
- セリフ: 「動くから動くんです！」
- 効果音: キラーン☆

# 5. 演出とスタイル (Direction & Style)
【ジャンル・雰囲気】ギャグ、一発ネタ
【画風】カラー、手書き風、シンプル、デフォルメ、表情豊か
【線画】シンプルで柔らかい線
【演出効果】表情の変化を強調、コマ3は開き直った爽やかさを表現

除外: 指が6本以上、不自然な体勢、コマごとの服装変化、顔が2つ
```
