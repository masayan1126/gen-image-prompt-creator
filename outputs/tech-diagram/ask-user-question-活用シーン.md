# AskUserQuestion の活用シーン

> **重要**: プロンプトを画像AIに入力する際は、必ずキャラクターの元画像を一緒に添付してください。

## プロンプト

```
【タイトル】AskUserQuestion の活用シーン（画像上部に大きく配置）

【図解タイプ】フローチャート
【テイスト】手書き風スケッチ、ラフな走り書き、ホワイトボードに描いたような、マーカーペンのタッチ、親しみやすい雰囲気
【カラースキーム】パステル調、白背景
【レイアウト】左から右への流れ、4ステップ

【解説キャラクター】添付画像のキャラクター（男の子とインコ）を右下に配置、フローを指し示すポーズ

【構成要素】
- ステップ1: タスク実行中に曖昧さを検出
- ステップ2: 質問を提示（選択肢付き）
- ステップ3: ユーザーが回答
- ステップ4: 回答を元に作業継続
- 各ステップ間は矢印で接続

【テキスト内容】
ステップ1: 「曖昧さを検出」 - タスク中に不明点を発見
ステップ2: 「質問を提示」 - 選択肢または自由入力で確認
ステップ3: 「ユーザーが回答」 - ワンクリックで選択
ステップ4: 「作業を継続」 - 正確な実装へ

【活用シーン例】（吹き出しで3つ配置）
- 「どのライブラリを使う？」（ライブラリ選択）
- 「認証方式はどちら？」（実装方針の決定）
- 「保存先はどこ？」（曖昧さの解消）

【スタイル指示】
- 手書き風スケッチ、ラフな走り書き、ホワイトボードに描いたような
- マーカーペンのタッチ、親しみやすい雰囲気
- 日本語テキスト
- 矢印は手描き風、流れが分かりやすく
- 各ボックスは角丸で柔らかい印象
- 吹き出しで活用シーン例を補足
- キャラクター（男の子とインコ）が解説している構図
```
